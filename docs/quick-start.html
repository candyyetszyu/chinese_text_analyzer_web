💻 System Requirements

Minimum Requirements
Python: 3.8 or higher
RAM: 2GB minimum, 4GB recommended
Storage: 1GB free space
OS: Windows 10+, macOS 10.14+, Ubuntu 18.04+
Recommended for Advanced Features
RAM: 8GB+ for large text processing
GPU: CUDA-compatible for GPU acceleration
Redis: For task queue functionality
Multi-core CPU: For parallel processing
Platform-Specific Notes
  • Windows: Use WSL2 for best compatibility
  • macOS: Install Xcode Command Line Tools
  • Linux: Install build-essential package
  • Docker: Supported via containerization

🛠️ Installation

Requirements: Python 3.8+ and pip package manager

Quick Setup (Recommended)

Step 1: Clone the Repository
git clone <repository-url>
cd chinese_text_analyzer_web
Step 2: Run Setup Script
python setup.py
This creates virtual environment and installs all dependencies
Step 3: Activate Virtual Environment
# On macOS/Linux:
source venv/bin/activate

# On Windows:
venv\Scripts\activate
Step 4: Run the Application
# Web interface:
python run_web.py

# CLI interface:
python run_cli.py

Manual Setup

If you prefer manual installation:

# Create virtual environment
python3 -m venv venv
source venv/bin/activate # On Windows: venv\Scripts\activate

# Install dependencies
pip install --upgrade pip
pip install -r requirements.txt

# Create necessary directories
mkdir -p data/input data/output/uploads src/web/static/results logs

🎯 First Run

Let's test your installation!

Starting the Web Interface

  1. Start the web server:
    python run_web.py
  2. Open your browser and navigate to:
    http://localhost:3000
  3. Test with sample text:
    現代科技改變了人們的生活方式，人工智慧發展迅速。
  4. Click "分析文本" (Analyze Text)

Testing the CLI Interface

  1. Start the CLI:
    python run_cli.py
  2. Follow the interactive menu
  3. Try analyzing the sample file:
    data/input/sample.txt
Quick Test API

Test the REST API with curl:

curl -X POST http://localhost:3000/api/analyze \
  -H "Content-Type: application/json" \
  -d '{"text": "這是一個測試"}'
